# Courses in LaTeX

This repository contains course material written in **LaTeX**, organized by subject and structured for clean compilation and editing. We aim to keep course notes consistent and easy to build into high-quality PDFs.

---

## Purpose

- Store course notes in a structured and version-controlled way.
- Write clean, professional documents using LaTeX.
- Keep a consistent format across multiple subjects.
- Make it easy to compile notes into PDFs for studying or sharing.

---

## Repository Structure
